Determine if an earthquake is happening based on internal accelerometer and gyroscope sensors in internet of things. (JSON object)

Types of Data Returned: Unstructured data, semistructured and structured data.

Schema on Write vs Read
  • Write- structured relational data

  • Read - write anything, load the data, then provide the user the ability to load the schema later.
      • Load data directly into HTFS (File system) and use map-reduce to extract the data out of JSON in more horizontal format

      accel _x _y _z Gyro _x _y _z values for each axis mapped to relation DB
